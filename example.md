# Exemplo 1: Markdown para PDF

Este exemplo mostra o que acontece quando você converte um arquivo `.md` para `.pdf`.

## Estrutura do documento

- Títulos em níveis diferentes (`H1`, `H2`, `H3`)
- Listas
- Tabelas
- Texto com destaque

### Por que tabelas ajudam

Quando você usa tabelas no Markdown, o conversor transforma isso em uma tabela no PDF (facilitando leitura no Google Drive ou no WhatsApp).

## Tabela comparativa

| Recurso | Onde aparece no PDF | Para que serve |
|---|---|---|
| Listas | com marcadores | organizar passos e bullets |
| Títulos | em hierarquia | facilitar escaneamento |
| Tabelas | no corpo do documento | comparar informações de um jeito claro |

## Pequena mensagem

Se você quiser, ajuste os textos e rode o comando para gerar o PDF novamente.
