# Exemplo 2: Títulos diferentes + mais detalhes

## Seções bem separadas

### Objetivo

Esse segundo exemplo é para testar variações de Markdown (tabela, negrito e itálico).

> Dica: este projeto roda localmente, então você pode converter seus próprios arquivos sem enviar dados para servidores.

## Mini checklist

- **Atualizar** os textos do seu roteiro
- Conferir a tabela
- Gerar o PDF de novo

## Tabela com “quebra de linha” visual

| Tipo de conteúdo | Exemplo | Observação |
|---|---|---|
| Texto explicativo | “Este PDF foi gerado a partir do Markdown.” | bom para compartilhar |
| Estrutura | H1, H2, H3 | ajuda quem lê no celular |
| Formatação | *itálico* e **negrito** | melhora a legibilidade |

### Fechamento

Se estiver pronto, execute o comando de conversão e abra o arquivo na pasta `docs/pdfs/`.
